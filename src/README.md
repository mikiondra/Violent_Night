# Violent Night
Is a short text-based adventure game developed in Java. You take role as Plirchin as he tries to get to lake to see Aurora borealis.

## How to play
You will be playing game by typing commands, these include:  

Drop - dropping items;

End - ending the game;

Explore - exploring locations, thus giving some flavour text;

Help - informing you about available commands;

Pick - picking items;

Plan - showing you the map of game;

Talk - talking to characters, thus getting information what to do, or getting flavour text;

Things - showing you the map of items in this game;

Tip - gives tip how to continue;

Use - using items;

Walk - walking between locations.

## Game mechanics
In this game you can walk between locations, pick, drop , use items you find, talk with characters, explore locations, use your map.  
Also, there are some small puzzles.

## How to start this game
Install zip file and open it in whatever Java program you have, even in notepad if you want. Everything must be there otherwise game can't be started.

## Libraries
Gson,newest version, from Google: for reading JSON file, where game data are located.
